# GC-Bridge-4

Django-based integration bridge project.

## Quickstart

1. Ensure Python 3.12+ is available.
2. Install dependencies: `uv sync` or `python -m pip install -e .`
3. Run migrations: `python manage.py migrate`
4. Start the dev server: `python manage.py runserver`

## Notes

- Configuration and secrets should live in a local `.env` file (not committed).
